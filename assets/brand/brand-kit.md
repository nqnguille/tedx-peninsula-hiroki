# TEDx Official Brand Kit — TEDxPeninsulaHiroki

## Colores oficiales

| Nombre | HEX | Uso |
|---|---|---|
| TED Red | `#EB0028` | "TEDx" wordmark — obligatorio |
| Negro | `#000000` / `#0A0A0A` | Fondos, texto sobre claro |
| Blanco | `#FFFFFF` | Texto sobre oscuro |

**Pantone**: 185 (spot color, offset printing)
**CMYK**: proceso 4 colores como alternativa

## Tipografía

- **Primary**: Helvetica Neue (pre-instalada en la mayoría de sistemas)
- **Alternativa open-source**: Inter Light (aprobada por TED como sustituto)
- **Letter-spacing**: 0 (no alterar)
- **"TEDx" weight**: Bold/700
- **Nombre del evento weight**: Light/300

## Reglas del logo

1. "TEDx" siempre en rojo `#EB0028`
2. Nombre del evento: negro o blanco únicamente
3. Fondo: solo blanco o negro sólido
4. "TEDx" y nombre del evento deben tener la misma altura visual
5. El nombre del evento se alinea a la izquierda del "TEDx"
6. Nunca usar el logo TED standalone
7. Nunca usar "TEDx" standalone — siempre con el nombre del evento

## Footer text requerido por TED

Texto obligatorio en todo sitio TEDx:
```
This independent TEDx event is operated under license from TED.
```

## Guidelines vigentes

- Versión 2023+ (mandatory para eventos nuevos desde enero 2023)
- Ya no es requerido incluir "x=independently organized TED event" en el logo
- Fuente: https://www.ted.com/pages/your-tedx-logo
